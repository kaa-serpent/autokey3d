How to Add a New Profile

Follow these steps to create an SVG keyway profile from a lock photo and import it into AutoKey 3D

1

Take a photo of the lock keyway

Place the lock face-up on a flat surface and take a clear, straight-on photo of the keyway. Make sure the image is sharp and well-lit — you will trace over it, so detail matters. Use a macro mode or get close enough so the keyway fills most of the frame.

Example: EVVA lock photographed straight-on

2

Open the photo in Inkscape (or any SVG editor)

Open Inkscape (free & open-source) or any vector graphics tool that can export SVG. Import your photo as a background layer: File → Import, then select your image. Lock the image layer so you don't accidentally move it while tracing.

The lock photo loaded as a reference layer in Inkscape

3

Trace the keyway profile and export as SVG

On a new layer, use the Bezier / Pen tool (or the Node tool) to draw the outline of the keyway. Trace the inner shape of the lock slot as a closed path — this silhouette is what AutoKey 3D uses to generate the key blank.

  • Keep only the path layer visible when you export — hide the photo layer.
  • The path must be a single closed shape filled with black.
  • Any reasonable scale works; the app normalises dimensions using the ph_base parameter.

When done: File → Save As and choose Plain SVG (or Inkscape SVG — both work).

Saving the traced keyway shape as an SVG file

4

Import the SVG in the Profiles tab

Go to Profiles → Add Profile. Fill in a name, upload your SVG, and set the required parameters:

  • Tolerance (tol) — extra clearance in mm so the key slides into the lock smoothly (start with 0.2).
  • Height base (ph_base) — the real-world height of the profile in mm (measure the keyway opening with calipers).

Click Save Profile and your new profile will appear in the list, ready to use when generating keys.
